### kicad获取库的方法
- 打开PCB编辑器，查看3D模型
- 选中想要的封装，按`E`编辑
- 获取库文件位置
    - 如下图所示
    ![](images/封装_3D获取.png)
- 找到对应文件，复制到自己的库文件路径下
    - 重命名规则，kicad有自带的封装，没有3D文件，3D文件命名应和库文件保持一致。
